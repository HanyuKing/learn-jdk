## 填写说明

1. **不需要修改**：可留空。
2. **建议改为别的英文**：在"产品建议"列填写期望的英文。
3. **建议拆分/合并**：如果同一行的多个中文其实表达不同含义，请在该行备注；反之若两行表达相同请也注明。
4. **建议下线**：若中文本身是测试/废弃数据（例如「外观test」「测试3」），可在该行写"下线"。
5. **促销文案占位符**：含 `%s` 的文案，请保持占位符位置；如需调整语序也请保留 `%s`，由开发同学按规则替换。

完成后回传给开发同学，我们会按建议更新翻译并刷新数据库。
